Le cours - les tests 

------------------------------------

SI condition
ALORS
    bloc d'instructions
SINON
    bloc d'instructions
FINSI

la condition est une expression logique donnant le résultat VRAI ou FAUX c'est un booléen 

------------------------------------

Les exercices - les tests

------------------------------------

ALGORITHME algoTest1

VARIABLE
    aLaClef : BOOL

DEBUT
    aLaClef <- FALSE
    avancer
    SI aLaClef = TRUE 
    ALORS 
        avancer
    SINON
        tournerDroite
        avancer
        aLaClef = TRUE
        tournerGauche
        avancer
        tournerDroite
        avancer
    FINSI
    avancer
FIN

ALGORITHME algoTest2

ALGORITHME algoTestExemple1

VARIABLE
    age : INT
    sexe : BOOL

DEBUT
    age <- ageDuPersonnage
    sexe <- sexeDuPersonnage
    SI age > 20 ET sexe = TRUE
    ALORS
        avancer
    SINON
        AFFICHER "je ne peux pas passer"
        arretDeLalgo
    FINSI
    avancer
FIN
